# Photographic Studio

Одностраничный сайт для фотографической мастерской в Санкт‑Петербурге.
Формат: один портрет, один кадр, медленный ритуал.

## Состав
- `index.html` — структура страницы
- `styles.css` — стиль (сепия, фактура, рамка)
- `script.js` — модальные окна
- `assets/background.jpg` — фон (сепия)
- `business-plan.md` — черновик бизнес‑плана
- `pitch-deck.md` — черновик презентации

## Запуск локально
Открой `index.html` в браузере.
